# Compound Annotation Database Selection: Formal Analysis

**Date**: 2026-04-05
**Context**: JUMP-CP batch correction paper — we need higher-hierarchy biological labels (mechanism of action, target) to evaluate whether batch correction preserves functional signal. This document provides a data-driven justification for database selection.

---

## 1. Problem Statement

Compound-level mAP (eval_key=`Metadata_JCP2022`) measures whether replicates of the same compound cluster together after batch correction. This is necessary but insufficient — a method could perfectly preserve compound identity while destroying the underlying biology (e.g., compressing all compounds into a single cluster).

**Functional signal evaluation** requires grouping compounds by their biological activity (MOA or target) and measuring whether compounds sharing a mechanism cluster together. This requires external annotation databases that map compound identities (InChIKey) to biological function.

**Requirements**:
- Labels must group compounds: need >= 3 compounds per group for robust mAP
- Multiple granularities desirable: target-level (specific) + MOA-level (broad)
- Coverage matters: more evaluable compounds = more statistical power
- Two compound sets to annotate:
  - **TARGET2** (302 compounds): well-characterized drugs, present in all sources
  - **COMPOUND** (~116K compounds): diverse screening library, present in subsets of sources

## 2. Candidate Database Profiles

### 2.1 Drug Repurposing Hub (DRH)

| Property | Value |
|----------|-------|
| **Maintainer** | Broad Institute (Corsello, Bittker, Golub) |
| **Publication** | Corsello et al., Nature Medicine 2017 |
| **Curation** | Expert-curated compound identities + literature-reported MOA/target |
| **Size** | ~6,700 compounds, 1,132 unique MOAs |
| **Identifiers** | InChIKey (direct match to JUMP-CP) |
| **Labels** | MOA (medium granularity, e.g. "CDK inhibitor") + Target (gene symbol) |
| **License** | Open (served via CLUE/S3) |
| **Version** | March 24, 2020 (frozen) |

**Coverage**:
- TARGET2: ~100%. After filtering >= 3 cpds/MOA: **31 MOAs, ~110 compounds**
- COMPOUND: 3,103 / 116K = **3.8%**. After >= 5 cpds/MOA: ~185 MOA classes

**Field adoption**: De facto standard for Cell Painting evaluation. Used by Chandrasekaran et al. 2024 (JUMP-CP dataset paper), Seal et al. 2024 ("Cell Painting: a decade of discovery"), and Arevalo et al. 2024 (the batch correction benchmark we are extending). Same institution as JUMP-CP.

**Already integrated**: `inputs/metadata/repurposinghub_inchi_to_moa_moaexploded.csv` (7,340 rows), `_load_repurposinghub_moa_info()` in `metrics/scib.py`, mAP computation enabled in `metrics/map.py`.

### 2.2 ChEMBL

| Property | Value |
|----------|-------|
| **Maintainer** | EMBL-EBI |
| **Publication** | Zdrazil et al., Nucleic Acids Research 2024 |
| **Curation** | Two tiers: DRUG_MECHANISM (expert-curated, ~5,392 cpds) + bioactivity (literature-extracted, ~2.5M cpds) |
| **Size** | 614,594 compound-target pairs from bioactivity; 5,392 curated MOA entries |
| **Identifiers** | ChEMBL ID + InChIKey (direct match) |
| **Labels** | Target (gene symbol) + action type (inhibitor/agonist) from DRUG_MECHANISM; target + activity value from bioactivity |
| **License** | CC BY-SA 4.0 (open) |
| **Version** | ChEMBL 35 (2024), continuously updated |

**Coverage**:
- TARGET2: near-complete (well-characterized drugs)
- COMPOUND: **72,654 unique InChIKeys = ~62%** (from bioactivity tier)
- After grouping by target at appropriate activity threshold: thousands of groups

**Existing in codebase**: `inputs/metadata/target_annotations.parquet` (590,042 rows, generated by `rules/consistency.smk` from CLUE/DRH target data). The current file uses DRH target labels — switching to ChEMBL bioactivity would dramatically expand coverage.

### 2.3 Open Targets Platform

| Property | Value |
|----------|-------|
| **Maintainer** | EMBL-EBI + Wellcome Sanger Institute |
| **Publication** | Ochoa et al., Nucleic Acids Research 2023 |
| **Curation** | Derives MOA from ChEMBL DRUG_MECHANISM table |
| **Size** | ~6,515 drug entries |
| **Identifiers** | ChEMBL ID (requires mapping) |
| **Labels** | MOA + target + disease indication |
| **License** | Open |

**Coverage**:
- TARGET2: **36 / 302 = 11.9%** (from `opentargets_moa_target2_eval.parquet`: 68 rows, 29 unique MOAs)
- COMPOUND: minimal (drug-disease focus, not compound-target)

**Already integrated**: `_load_opentargets_moa_info()` in `metrics/scib.py`, but produces only 36 evaluable compounds — insufficient for statistical analysis.

### 2.4 RefChemDB

| Property | Value |
|----------|-------|
| **Maintainer** | US EPA |
| **Publication** | Judson et al. 2019 |
| **Curation** | Semi-automated aggregation from 16+ public sources with evidence counts |
| **Size** | ~41K chemicals |
| **Identifiers** | DTXSID (requires two-hop bridge: InChIKey -> PubChem CID -> DTXSID) |
| **Labels** | Target + mode (agonist/antagonist/inhibitor) |
| **License** | Open |

**Coverage** (evaluated in `exploration/refchemdb_mapping.py`):
- TARGET2 at evidence >= 2: **40 / 302 = 13.2%**, yielding **2 evaluable targets**
- TARGET2 at evidence >= 1: 113 / 302, 77 evaluable (but single-source = 39.5% precision per Judson et al.)
- COMPOUND: not evaluated, expected low (environmental tox chemical space)

**Root cause of poor coverage**: Chemical space mismatch. RefChemDB is focused on environmental/industrial toxicology targets (AR, ESR1, ACHE, DRD2). JUMP-CP TARGET2 compounds are predominantly kinase inhibitors and oncology drugs with minimal overlap.

### 2.5 DrugBank

| Property | Value |
|----------|-------|
| **Maintainer** | University of Alberta |
| **Publication** | Knox et al., Nucleic Acids Research 2024 |
| **Curation** | Expert-curated, highest quality |
| **Size** | ~10,800 drugs (4,563 approved + 6,231 investigational) |
| **Identifiers** | DrugBank ID + InChIKey |
| **Labels** | MOA + target + pathway + pharmacological action (rich hierarchy) |
| **License** | **CC BY-NC 4.0** (academic use requires license application) |

**Coverage**: Similar to DRH for TARGET2 (well-characterized drugs). Very low for COMPOUND (drugs only).

### 2.6 Other Databases Considered

| Database | Why considered | Why rejected |
|----------|---------------|-------------|
| **PubChem BioAssay** | 286M substances, highest coverage | Too noisy (HTS active/inactive calls, high FPR). No curated MOA. Not used in any Cell Painting evaluation. |
| **CLUE / Connectivity Map** | Same institution as JUMP-CP | Annotation data IS the DRH data. No additional coverage. |
| **MOTIVE** (Arevalo et al., NeurIPS 2024) | Built by JUMP-CP team, aggregates 7 meta-databases | 3,632 compounds total. Built for DTI prediction, not evaluation labels. Worth citing but not a primary source. |
| **STITCH** | 300K+ chemicals with interaction scores | Requires confidence score filtering. No curated MOA hierarchy. |
| **BindingDB** | 1.4M compounds with binding measurements | Patent-derived. No MOA hierarchy. Requires activity thresholding. |

## 3. Quantitative Comparison

### 3.1 TARGET2 Coverage (302 compounds)

| Database | Compounds mapped | Evaluable groups (>= 3 cpds) | Evaluable compounds | Label type |
|----------|-----------------|------------------------------|--------------------:|------------|
| **DRH** | ~302 (100%) | 31 MOAs | ~110 | MOA |
| **DRH** | ~302 (100%) | ~60 targets | ~180 | Target |
| **ChEMBL** (DRUG_MECHANISM) | ~280 (93%) | ~50 targets | ~200 | Target + action |
| **ChEMBL** (bioactivity) | ~290 (96%) | ~80 targets | ~250 | Target (activity-based) |
| **Open Targets** | 36 (12%) | ~10 MOAs | ~30 | MOA |
| **RefChemDB** (count >= 2) | 40 (13%) | 2 targets | ~8 | Target + mode |
| **DrugBank** | ~280 (93%) | ~50 targets | ~200 | Target + pathway |

### 3.2 COMPOUND Library Coverage (~116K compounds)

| Database | Compounds mapped | Coverage | Evaluable potential |
|----------|-----------------|---------|-------------------|
| **ChEMBL** (bioactivity) | ~72,654 | **62%** | Thousands of target groups |
| **DRH** | ~3,103 | 3.8% | ~185 MOA groups |
| **MOTIVE** | ~3,632 | 3.1% | Limited |
| **Open Targets** | <1,000 | <1% | Insufficient |
| **RefChemDB** | Not evaluated | Expected <5% | Chemical space mismatch |
| **DrugBank** | <5,000 | <5% | Drugs only |

### 3.3 Key Insight

No single database provides both high-quality MOA labels AND broad COMPOUND coverage. The databases stratify into two tiers:

- **High-quality, low-coverage**: DRH, ChEMBL DRUG_MECHANISM, DrugBank (~5-10K compounds, expert-curated MOA)
- **Broad-coverage, target-level**: ChEMBL bioactivity (~72K compounds, requires activity thresholding)

## 4. Recommendation: Two-Tier Annotation Strategy

### Tier 1: Drug Repurposing Hub (Primary, Curated MOA)

**Use for**: MOA-level mAP evaluation. Primary metric for comparing batch correction methods on functional signal preservation.

**Justification**:
1. **Field standard**: Every published Cell Painting benchmark uses DRH (Arevalo 2024, Chandrasekaran 2024, Seal 2024). Using the same source ensures direct comparability.
2. **Same institution**: DRH is maintained by the Broad Institute, the same group that produced JUMP-CP. Compound identifiers and naming conventions are aligned.
3. **Already integrated**: Loader functions, annotation files, and mAP computation infrastructure exist in the codebase.
4. **High curation quality**: Expert-curated MOA labels with experimentally confirmed compound identities.
5. **TARGET2 coverage**: Near-complete, yielding 31 evaluable MOA classes — sufficient for robust mAP.

**Limitation acknowledged**: 3.8% coverage of COMPOUND library limits evaluation on larger scenarios. This is a known limitation shared by all published benchmarks.

**Eval key**: `Metadata_DRH_MOA` (MOA-level), `Metadata_DRH_TARGET` (target-level)

### Tier 2: ChEMBL Bioactivity (Broad Coverage, Target-Level)

**Use for**: Target-level mAP evaluation on COMPOUND plates. Extends functional evaluation beyond the 302 TARGET2 compounds to the full screening library.

**Justification**:
1. **Coverage**: 62% of the ~116K COMPOUND library — 20x more compounds than DRH. This is the only database that can provide meaningful evaluation on COMPOUND plates.
2. **Direct InChIKey matching**: No identifier bridging needed (unlike RefChemDB's two-hop InChIKey -> CID -> DTXSID).
3. **Partially integrated**: `inputs/metadata/target_annotations.parquet` exists (590K rows). The consistency pipeline (`rules/consistency.smk`) already computes target-level mAP using this data.
4. **Complementary granularity**: Target-level labels (e.g., "EGFR", "BRAF") are finer-grained than MOA labels (e.g., "EGFR inhibitor"). This tests whether methods preserve target-specific signal, not just broad mechanism.
5. **Open license**: CC BY-SA 4.0.

**Activity threshold**: Compounds are considered active against a target if they have a measured pChEMBL value >= 6 (1 uM) in at least one assay. This follows the community standard for defining pharmacologically relevant activity (Zdrazil et al. 2024). Sensitivity analysis at pChEMBL >= 5 (10 uM) and >= 7 (100 nM) thresholds will be reported in supplementary.

**Eval key**: `Metadata_ChEMBL_TARGET` (target-level)

### Explicitly Rejected

| Database | Reason |
|----------|--------|
| **RefChemDB** | Chemical space mismatch (environmental tox vs oncology). 2 evaluable targets at quality threshold. Evaluated in `exploration/refchemdb_mapping.py`. |
| **Open Targets** | 12% TARGET2 coverage — strictly inferior to DRH which covers 100%. MOA data is a subset of ChEMBL. |
| **DrugBank** | CC BY-NC license creates friction. Similar coverage to DRH. Not the field standard. |
| **PubChem BioAssay** | No curated labels. HTS active/inactive calls have high false-positive rates. Not used in any published Cell Painting evaluation. |

## 5. Evaluation Design

### 5.1 Three Complementary Evaluation Levels

| Level | Eval key | Source | What it measures | Expected scenarios |
|-------|----------|--------|-----------------|-------------------|
| **Compound** | `Metadata_JCP2022` | JUMP metadata | Replicate reproducibility | All (S1-S5+) |
| **MOA** | `Metadata_DRH_MOA` | Drug Repurposing Hub | Mechanism-level signal | All (31 MOAs on TARGET2; ~185 on COMPOUND) |
| **Target** | `Metadata_ChEMBL_TARGET` | ChEMBL bioactivity | Target-specific signal | COMPOUND-plate scenarios (S3, S5, wave2) |

### 5.2 Expected Statistical Power

| Scenario | Plates | Compound mAP (n cpds) | MOA mAP (n groups) | Target mAP (n groups) |
|----------|--------|----------------------|--------------------|--------------------|
| S1 (1 src, T2) | TARGET2 | 302 | 31 MOAs (~110 cpds) | ~60 targets (~180 cpds) |
| S2 (3 src, T2) | TARGET2 | 302 | 31 | ~60 |
| S3 (3 src, T2+C) | Both | ~82K | 31 + ~100 | ~60 + thousands |
| S4 (5 src, T2) | TARGET2 | 302 | 31 | ~60 |
| S5 (5 src, T2+C) | Both | ~82K | 31 + ~100 | ~60 + thousands |

### 5.3 Multi-MOA Compound Handling

7.4% of DRH compounds have multiple MOA annotations (e.g., "CDK inhibitor; HDAC inhibitor"). Strategy:
- Use `copairs.map.multilabel` which natively handles multi-label mAP
- Already implemented in `metrics/map.py` via the `_merge_with_duplication()` function
- Report sensitivity to including vs excluding multi-MOA compounds

## 7. Paper Text (Draft)

> **Functional signal evaluation.** We evaluated whether batch correction preserves biological signal at two complementary granularities using external compound annotations.
>
> For mechanism-of-action (MOA) evaluation, we used the Broad Institute Drug Repurposing Hub (DRH; Corsello et al. 2017), which provides expert-curated MOA labels for clinically advanced compounds. DRH is the de facto standard for Cell Painting functional evaluation, used by the original JUMP-CP dataset characterization (Chandrasekaran et al. 2024) and the batch correction benchmark we extend (Arevalo et al. 2024). On TARGET2 plates (302 compounds present in all sources), DRH yields 31 evaluable MOA classes covering 110 compounds at a threshold of >= 3 compounds per class. On COMPOUND plates, DRH coverage is 3.8% — a known limitation shared by all published benchmarks.
>
> To extend evaluation to the full COMPOUND library (~116K compounds), we used target-level annotations from ChEMBL bioactivity data (Zdrazil et al. 2024), which covers 62% of COMPOUND compounds via direct InChIKey matching. Compounds were grouped by their primary pharmacological target at a pChEMBL >= 6 (1 uM) activity threshold. This complements MOA evaluation by testing whether finer-grained target-specific signal is preserved across batches.
>
> We considered but rejected several alternative databases: RefChemDB (Judson et al. 2019) due to chemical space mismatch with JUMP-CP compounds (2 evaluable targets at quality threshold); Open Targets (Ochoa et al. 2023) due to low coverage (12% of TARGET2, strictly inferior to DRH); and DrugBank (Knox et al. 2024) due to licensing constraints and similar coverage to DRH.
>
> For both annotation tiers, we computed median average precision (mAP) using copairs (Arevalo et al. 2024), where positive pairs are compounds sharing the same MOA/target and negative pairs are compounds with different annotations. Multi-MOA compounds (7.4% of DRH) were handled natively via multilabel mAP computation.
